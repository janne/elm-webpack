Elm Webpack Boilerplate
=======================

Cloned and slightly modified example from https://github.com/rtfeldman/elm-webpack-loader

Setup
-----

```
$ npm install
$ elm-make
```

Develop
-------

```
$ npm start
$ open localhost:3000
```

Distribute
----------

```
$ npm run build
```

Open the `dist/index.html` file to see the result.
